About the App Border Delay Forecaster

This application was built as a capstone project for the Galvanize Data Science Bootcamp in Seattle. Using historical data since 2007 from US and Canadian border agencies, I used a decision tree machine learning model to predict wait times from 2014 through 2016.

The model predictions are labeled as predictions in the charts. Predictions from a naive model taking a 12-month average by day of week is labeled as baseline.

Details of the implementation can be found in my GitHub repo.
